Introducción Boostrap Table - Print
Para añadir la extensión de impresión debemos añadir este fichero de JavaScript 
<script src="https://unpkg.com/bootstrap-table@1.18.2/dist/extensions/print/bootstrap-table-print.min.js"></script>
Con este y con las librerías de Boostrap-Table ya podemos imprimir las tablas.
HTML
<table id="table" data-show-print="true" data-url="./json/data1.json">
    <thead>
    <tr>
        <th data-field="id">ID</th>
        <th data-field="name">Item Name</th>
        <th data-field="price">Item Price</th>
        <th data-field="amount">Amount</th>
    </tr>
    </thead>

</table>
Javascript
var $table = $('#table');
$table.bootstrapTable();
ID Item Name Item Price Amount

Introducción Boostrap Table - Stycky Header
Para usar esta extensión debemos incluir las siguientes librerías 
<link rel="stylesheet" href="https://unpkg.com/bootstrap-table@1.18.2/dist/extensions/sticky-header/bootstrap-table-sticky-header.css">
<script src="https://unpkg.com/bootstrap-table@1.18.2/dist/extensions/sticky-header/bootstrap-table-sticky-header.js"></script>
añadiendo el siguiente código ya funcionaría
HTML
<table id="table"
       data-sticky-header="true"
       data-sticky-header-offset-left="1"
       data-sticky-header-offset-right="1"
       data-url="./json/data1.json">
    <thead>
    <tr>
        <th data-field="id">ID</th>
        <th data-field="name">Item Name</th>
        <th data-field="price">Item Price</th>
        <th data-field="amount">Amount</th>
    </tr>
    </thead>

</table>

var $table = $('#table');

Introducción Boostrap Table - Multi Sort
Librería necesaria para el uso
<script src="https://unpkg.com/bootstrap-table@1.18.2/dist/extensions/multiple-sort/bootstrap-table-multiple-sort.js"></script>

HTML
<table id="table"
        data-show-multi-sort="true"
        data-sort-priority='[{"sortName": "id","sortOrder":"desc"},{"sortName":"name","sortOrder":"desc"}]'
        data-url="./json/data1.json">
    <thead>
    <tr>
        <th data-field="id" data-sortable="true">ID</th>
        <th data-field="name" data-sortable="true">Item Name</th>
        <th data-field="price" data-sortable="true">Item Price</th>
        <th data-field="amount" data-sortable="true">Amount</th>
    </tr>
    </thead>

</table>
Javascript
var $table = $('#table');
$table.bootstrapTable();
ID Item Name Item Price Amount
